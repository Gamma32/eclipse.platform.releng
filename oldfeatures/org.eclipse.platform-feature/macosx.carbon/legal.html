Locating License Agreements and Notices

10th September, 2002

This folder should contain the following files:

  • notice.html - The Eclipse.org Software User Agreement.
  • cpl-v10.html - The Common Public License v1.0 (referenced in the Software User Agreement).
  • Eclipse_2.0.app - The Eclipse application bundle. The bundle may contain other license agreements and notices as referred to in the Software User Agreement.

The Mac Finder will display the Eclipse application bundle as a single icon in the desktop. To locate the other license agreements and notices contained in the application bundle:

  1. Select the Eclipse application bundle icon while holding down the Control Key. This will present you with a popup menu.
  2. Select "Show Package Contents" in the popup menu.
  3. Locate license agreements and notices (described in the Software User Agreement) in the application bundle folders.
